# 🤖 AI Calculator using Flask

A modern AI-themed Calculator built using **Python** and **Flask**. This project performs basic arithmetic operations through a web interface with a futuristic dark UI design.

## 🚀 Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* AI-inspired dark theme UI
* Responsive and user-friendly interface
* Built with Flask and HTML/CSS

## 🛠️ Technologies Used

* Python
* Flask
* HTML5
* CSS3

## 📂 Project Structure

```text
Py Arithmetic Calculator/
│
├── app.py
│
└── templates/
    └── index.html
```

## ⚙️ Installation

1. Clone the repository:

```bash
git clone <your-repository-url>
```

2. Navigate to the project folder:

```bash
cd Py Arithmetic Calculator
```

3. Install Flask:

```bash
pip install flask
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```text
http://127.0.0.1:5000
```
